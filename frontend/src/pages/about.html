WEB PROGAMMING

Get to know the basics of website build !!

Description Introduction Syllabus Content Review FAQ

Every first start,

Web programming is a term closely related to websites and the internet. Why is that? Because web programming is one of the processes of creating websites for internet purposes, commonly referred to as the **WWW** or **World Wide Web**. The term WWW is widely known because it is the most popular internet service today.

In this course, you will learn how to create websites to industry standards. Here, you will be taught **HTML**, **CSS**, and **JavaScript**, which are the basic foundations of website creation.

The benefits of learning website programming are:
* Ability to run application and software businesses
* Ability to build your own website
* Supporting your career

Target and Goal

The class is intended for students with a high level of curiosity in the field of website programming.
This class provides a certificate when students can complete each course they study.
It is expected that students can complete all the assignments given at the end of the course.
This class is designed for beginners so there are no prerequisites in understanding programming beforehand.

Specifications & Software

**PC/Laptop** with minimum **4 GB RAM** and **128 GB free disk**
**Browser**
**Text Editor** (VsCode,...)
**Internet access**
